# 🐍 Si Ular Cerdas (Smart Snake)

Si Ular Cerdas adalah permainan ular tangga interaktif edukatif yang menggabungkan keseruan *board game* klasik dengan tantangan berbasis matematika. Proyek ini dikembangkan secara *end-to-end* untuk memberikan pengalaman belajar yang menyenangkan dan interaktif.

## 🛠️ Tech Stack & Tools
- **Game Engine:** Unity
- **Backend/Database:** Firebase
- **Language:** C# (C-Sharp)

## ✨ Key Features
- **Interactive Math Challenges:** Mengintegrasikan soal-soal matematika yang harus diselesaikan pemain untuk maju di papan permainan.
- **Dynamic User Interface:** Antarmuka visual yang menarik dan disesuaikan untuk pengalaman bermain yang lancar.
- **Real-time Data Integration:** Menggunakan Firebase untuk manajemen data (misalnya: skor, profil pemain, atau *leaderboard*).
- **Cross-Platform Potential:** Dikembangkan menggunakan Unity, memungkinkan skalabilitas untuk berbagai *platform*.
